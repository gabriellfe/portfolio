    Microserviços Java

    Desenvolvi um ecosistema de Microserviços de 6 serviços sendo Login, Ordem, Pagamento, Produto, Gateway, Config Server. Onde a segurança é feita pelo Gateway atráves de Filters aplicados em cada rota e as confirações ficam alocadas no config server sendo possível fazer o refresh, sem precisar do redeploy. Possui também um projeto utilizado como Commons, sendo importado pelos demais com a responsabilidade de unificar as configurações principais de conexão
